# Investment Dashboard

A responsive, self-contained investment dashboard for portfolio monitoring. It includes portfolio KPIs, performance comparison, asset allocation, holdings, watchlist, and risk indicators.

## Open locally

Open `index.html` in a browser. No build step is required.

## Files

- `index.html` - dashboard markup
- `styles.css` - layout and visual styling
- `script.js` - dashboard data rendering and chart drawing
